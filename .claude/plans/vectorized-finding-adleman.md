# 홍보현황 업데이트: 소규모/수도/학교 3탭

## Context
현재 홍보현황은 소규모(15시)/수도(7시) 2탭. 학교 탭 추가 요청 + 각 시 안에 동 리스트 필요. 학교는 NEIS API로 수원/화성/안양 초중고 데이터 가져와서 동별 분류.

## 변경 사항

### 1. 3탭 구조
- 소규모 (15시) → 시 클릭 시 동 리스트 펼침
- 수도 (7시) → 시 클릭 시 동 리스트 펼침
- 학교 (수원/화성/안양) → 동별 학교 리스트 (초/중/고 분류)

### 2. 시 → 동 리스트 (소규모/수도)
- 기존: 시별 방문 현황만 표시
- 변경: 시 클릭 → 아코디언으로 동 리스트 펼침
- 각 동: 방문 횟수, 최근 방문일, 담당자

### 3. 학교 탭
- NEIS API로 수원/화성/안양 학교 데이터 로드
- API route: `/api/schools` (NEIS_API_KEY 사용)
- 시 → 동 → 학교 리스트 (초/중/고 아이콘 구분)
- 학교별 홍보 기록 연동 (promo_records category='학교')

### 4. 파일 수정
- `src/components/calendar/WorkCalendarPage.tsx` — PromoStatusTab 컴포넌트
- `src/app/api/schools/route.ts` — 신규 API route (NEIS 학교 조회)

### 5. 검증
- 소규모/수도 탭: 시 클릭 → 동 리스트 펼침
- 학교 탭: 수원/화성/안양 동별 학교 목록
- 빌드 성공
