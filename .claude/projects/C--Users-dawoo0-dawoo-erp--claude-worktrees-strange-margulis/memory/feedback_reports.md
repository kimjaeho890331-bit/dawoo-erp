"숫자만으로 판단하려고 온게아니야. 글로 적어주고 상황에 대해서 설명을 해야지"
**Why:** 사장님은 데이터를 직접 해석하는 게 아니라, AI가 상황 분석 + 판단 + 조치 방향까지 서술해야 함.
**How to apply:**
- 보고서는 "사장님, OO 보고 드립니다" 형식의 서술체
- 긴급/주의 사항은 빨간/노란 블록으로 분리
- 각 섹션은 문단으로 상황 설명 + 원인 분석 + 조치 제안
- 숫자는 문장 안에 녹여서 맥락과 함께 제시
- 마무리: "이상 보고 드렸습니다. 추가 지시 부탁드립니다."
