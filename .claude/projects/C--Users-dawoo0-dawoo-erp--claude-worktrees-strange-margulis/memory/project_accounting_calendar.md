사장님이 코스카 세무달력 사진(2026년 6~10월) 제공. 회계달력(/accounting-cal) 페이지에 반영할 데이터.

**Why:** 건설업 세무 신고/납부 기한을 놓치면 가산세 발생. 회계달력에 미리 표시하여 일정 관리 필수.

**How to apply:**
- 회계달력 페이지 개발 시 이 데이터를 기반으로 반복 일정을 자동 생성
- 매년 같은 규칙 적용 가능 (법정 기한이므로)
- 주말/공휴일 보정 로직 필요 (다음 영업일로 이동)

## 매월 반복 (Monthly Recurring)

| 일자 | 항목 |
|------|------|
| ~5일 | 일용근로자 국민연금·건강보험 자격변동 신고 |
| ~9일 | (건설현장 사업장) 국민연금·건강보험료 수시 경정고지 재산정 |
| 10일 | 원천세·주민세(종업원분) 신고 납부 / 국민연금·건강보험료 납부 |
| 15일 | 고용·산재 근로내용 확인신고 |
| 말일 | 일용근로소득 지급명세서 제출(전월분) / 간이지급명세서(근로소득, 거주자의 사업소득) 제출 |

## 분기별 (Quarterly)

| 월 | 일자 | 항목 |
|----|------|------|
| 1월 | 25일 | 부가가치세 2기 확정신고 납부 |
| 4월 | 25일 | 부가가치세 1기 예정신고 납부 |
| 7월 | 25일 | 부가가치세 1기 확정신고 납부 |
| 10월 | 25일 | 부가가치세 2기 예정신고 납부 |

## 반기별 (Semi-Annual)

| 월 | 항목 |
|----|------|
| 1월 | 원천세 반기납부(하반기분) |
| 6월 | 재산세(주택분 1/2, 토지분) 납부 / 자동차세 1기분 납부 |
| 7월 | 원천세 반기납부(상반기분, 반기납 포함) / 재산세(주택분 1/2, 주택 이외 건축물) / 자동차세 납부 |
| 9월 | 원천세 반기납부 법인 하반기분 신청기한 |

## 연간 특수 일정 (Annual)

| 월 | 항목 |
|----|------|
| 6월 | 시공능력평가 신청기간 / 성실신고확인대상사업자 종소세 신고 |
| 7월 | 시공능력평가 공시 |
| 8월 | 시공능력평가 재측정기한 |
| 9월 | 근로장려금 반기분 신청기한 / 종합부동산세 과세특례 신고 |

## 주말/공휴일 보정 규칙
- 신고/납부 기한이 토·일·공휴일이면 → 다음 영업일로 자동 연장
- 예: 2026년 7/25(토) → 7/27(월), 10/25(일) → 10/26(월)
