## 핵심 철학

1. **AI가 주인공, 사람이 서브** — AI가 데이터 입력/서류 생성/판단. 사람은 확인/수정만.
2. **사람 화면은 사람 편의성 최우선** — AI가 아무리 잘해도 사람이 보조할 화면은 직관적이어야.
3. **단순한 구조** — 테이블 최소화. projects 한 테이블에 수도/소규모 항목 전부. AI가 한 테이블만 보는 게 실수 적음.
4. **이중 단계** — 화면 4단계(접수→승인공사→완료서류→수금) / AI 내부 10단계(문의→실사→견적→동의서→신청서제출→승인→착공계→공사→완료서류→입금).
5. **데이터 하나, 뷰 여러 개** — schedules DB 1개 → 캘린더 4곳에서 표시.
6. **드래그앤드롭 통일** — 모든 서류/사진 업로드는 드래그앤드롭 + 미리보기.
7. **활동 로그 자동** — 시스템 사용 = 업무 기록. 별도 보고서 작성 없음.
8. **에이전트에 회사를 넣는다** — 우리회사를 에이전트 안에 넣는 구조. 직원이 적으니까 에이전트가 회사를 이해하는 게 맞음.

## 하지 않는 것

- 승인번호 없음 / 지점 없음(수원본사만) / 현장 캘린더 별도 메뉴 없음
- 같은 주소 중복 차단 안 함(수도+소규모 동시 가능) / 면/리 주소도 저장 가능
- CRM(잠재고객) 별도 메뉴 없음 → 접수대장에 "문의(예약)" 상태로 포함
- 단가표 별도 페이지 없음(견적서 내장) / 접수대장에 시 컬럼 없음(태그 필터)
- 1차 개발 없음(전체 설계 후 한번에 제대로) / 입찰정보 수집 안 함(외부 업체)
- 연와조 등 건축 구조 정보 저장 안 함 / 자재단가 별도 페이지(코드에 녹임)

## 전체 데이터 흐름

### 캘린더 데이터 흐름
```
[입력 4곳]                         [DB 1개]           [출력 4곳]
접수대장 (실측/시공/제출일)    ──→               ──→  업무 캘린더
현장관리 (착공~준공 공정)      ──→   schedules   ──→  현장관리 캘린더(개별)
직원/관리자 (개인 일정)        ──→    테이블     ──→  접수대장 상세(해당건)
AI 비서 (홍보/미팅 제안)       ──→               ──→  대시보드(오늘 일정)

[AI 4기능]
① 빈 일정 감지 → 홍보/미팅 제안
② 일정 충돌 체크 → 협력업체 겹침 알림
③ 공문 패턴 예측 → 작년 기준 2달 전 알림 (늦으면 안 됨)
④ 마감 알림 → 제출 D-day
```

### 보고서/분석 데이터 흐름
```
[12개 소스]
접수대장(건수)     현장관리(공정/일지)   캘린더(준수율)      활동로그(직원사용)
지출결의서(항목별)  카드지출(CSV분석)     고정지출(고정비)    거래처DB(업체평가)
홍보현황(방문기록)  A/S관리(하자패턴)    미수금(완료건기준)   현장보고서(소장최종)
                              ↓ 전부
                        [AI 분석 엔진]
                              ↓
[4종 보고서]
매일: 직원 활동 요약 + 긴급 알림
주간: 지원사업 + 현장 공정 + 직원 성과 + 미수금 + 통합 요약
월간: 카드/고정 지출(새는돈 찾기) + KPI + 지원사업 통계 + 통합 요약
현장마감: 최종 보고서 + 수익률(마감기준) + 협력업체 평가
                              ↓
                     [총괄 보고서] 긴급→주의→정상 정렬
                              ↓
                   관리자 분석 페이지 + KPI
```

## AI 에이전트 구조 → docs/AGENT.md

```
사용자 입력 (웹 사이드패널)
    ↓
  라우터 (의도 파악)
    ├── 접수 에이전트: DB 등록/수정, 표제부/전유부 API 연동
    ├── 서류 에이전트: 서류함 템플릿 → 폼필드 데이터 입력 → PDF 생성
```

## 개발 순서

1. Supabase DB + Auth (로그인/권한: 관리자 vs 직원)
2. 레이아웃 + 사이드바 (16메뉴 7그룹) + AI 사이드 패널
3. 접수대장 (핵심)
4. 서류함 + PDF 자동화 (폼필드)
5. 견적서 (웹 스프레드시트)
6. 현장관리 (아코디언 + 공정 캘린더)
7. 업무 캘린더 + AI 교정 + 홍보현황
8. 대시보드 (직원용) + 관리자 분석 + 보고서 + KPI
9. 나머지 (지출결의서, 연차, 거래처, A/S, 카드분석, 고정지출, 회계달력, 공지, 직원관리)
