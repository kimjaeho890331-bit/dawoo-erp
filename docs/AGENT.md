# docs/AGENT.md - AI 에이전트 시스템 총괄

> 에이전트 상세 스펙은 팀별 MD를 참조하세요.
> 이 파일은 라우터 + 팀 구조 + 공통 규칙만 다룹니다.

## 전체 구조

```
사용자 입력 (웹 사이드패널)
    ↓
  라우터 (의도 파악 + 분배)
    ├── 접수팀 → docs/AGENT_REGISTER.md
    ├── 서류팀 → docs/AGENT_DOCS.md
    └── 조회팀 → docs/AGENT_QUERY.md
```

---

## 라우터 판단 규칙

### 접수팀으로 보내는 경우
- "XX빌라 접수해줘" / "접수" / "등록" / "추가"
- "단계 변경" / "승인" / "상태 변경"
- 고객 데이터 수정 관련

### 서류팀으로 보내는 경우
- "신청서 만들어줘" / "서류" / "PDF" / "견적서"
- "완료보고서" / "착공계"
- 템플릿/양식 관련

### 조회팀으로 보내는 경우
- "알려줘" / "현황" / "몇 건" / "일정"
- "보고서" / "KPI" / "분석"
- "미수금" / "이번 주" / "지난 달"

### 애매한 경우
- 되물어보기: "접수를 원하시나요, 조회를 원하시나요?"
- 복합 요청: 순서대로 처리 (예: "접수하고 신청서 만들어줘" → 접수팀 → 서류팀)

## 공통 판단 규칙

### AI가 혼자 해도 되는 것
- 표제부/전유부 자동 조회 → 면적, 세대수, 주소, 소유주 입력
- 데이터 채워지면 단계 자동 전환
- 데이터 충분하면 서류 자동 생성
- 활동 로그 자동 기록
- 보고서/KPI 자동 산출
- 캘린더 일정 자동 등록 (접수 시 실측일 등)

### 반드시 사람한테 물어봐야 하는 것

#### 접수 관련
- 같은 이름 빌라가 2개 이상 → "어떤 건물인가요?" + 주소 제시
- 주소 없이 접수 요청 → "주소를 알려주세요" (주소 필수)
- 공사 종류 미지정 → "어떤 공사인가요? 소규모/수도?" 물어보기
- 소규모인지 수도인지 애매 → **반드시 물어보기**
- 표제부 소유주 ≠ 접수 소유주 → "명의가 다릅니다" 알림

#### 빌라명 판단
- 사용자 "매산빌라" vs 표제부 "매산 하이츠빌" → 앞글자(대표명) 일치 → "정식 명칭은 '매산 하이츠빌'입니다. 맞나요?" 확인
- 완전 불일치 → "해당 주소의 건물명은 'OO'입니다. 맞는 주소인가요?"
- 확인 후 → 접수대장에 정식 명칭 저장 + 별칭 메모 기록 (검색 가능)

#### 공사종류 자동 분류
- 소규모 접수대장 → 방수, 옥상, 기와, 도장, 계단, 담장, 기타
- 수도 접수대장 → 수도, 공용, 아파트수도, 아파트공용, 옥내
- 모르는 종류 → 소규모 목록에서 선택지 제공
- 소규모/수도 구분 불가 → 반드시 물어보기

#### 금액 관련
- 총공사비 확정 → 사람 확인 필수
- 단가 변경 → "기존 견적서에도 영향 갑니다" 확인
- 견적서 금액 조정 → "어떤 항목 줄일까요?" 물어보기

#### 단계 관련
- 필수 항목 미입력 시 → 다음 단계 차단 + "XX 부족합니다" 구체적 안내
- 단계 건너뛰기 요청 → "현재 OO 단계인데 XX로 바꿀까요?" 확인
- 서류 데이터 부족 시 → 프린트/생성 차단 + 부족 항목 안내

### 절대 하면 안 되는 것
- 데이터 삭제 (사람만 가능)
- 확인 없이 서류 외부 제출/발송
- 추측으로 데이터 입력 (모르면 물어보기)
- 금액을 AI가 임의로 변경
- 다른 직원의 활동 로그를 해당 직원에게 공개

## AI 비서 사용 시나리오

### 접수
- "호계동 960번지 방수 접수해줘" → 소규모 접수대장 등록 + 표제부 자동 조회
- "분당 래미안 신청서 만들어줘" → 데이터 확인 → PDF 생성 or 부족 안내
- "XX빌라 승인 났어" → 승인일 입력 + 단계 전환

### 조회
- "이번 주 실측 일정" → 캘린더 조회 → 요약
- "미수금 현황" → 완료건 기준 미수금 집계
- "김직원 이번 주 뭐 했어?" → 활동 로그 요약
- "지난 달 카드 지출 이상한 거?" → 이상치 감지 결과

### 현장
- "호계 공정률?" → 65% + 미착수 공정 안내
- "산본 서류 뭐 빠졌어?" → AI 서류 체크 결과

### 일정
- "내일 오전 비어있는 직원?" → 캘린더 빈 일정 조회
- "수원 세류동 홍보 언제 갔었어?" → 방문 기록 조회
