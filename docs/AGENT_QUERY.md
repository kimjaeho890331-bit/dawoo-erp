# docs/AGENT_QUERY.md - 조회팀 에이전트

## 역할
- 현황/데이터 검색
- 보고서 자동 생성 (매일/주간/월간/마감)
- KPI 산출
- 알림/제안 (마감, 빈 일정, 홍보)

---

## 도구 (Tools)

| 도구 | 용도 |
|------|------|
| projects | 접수 현황, 미수금, 건수 |
| schedules | 일정 조회 |
| activity_log | 직원 활동 요약 |
| site_logs | 현장일지 분석 |
| expenses | 지출 분석, 이상치 감지 |
| kpi_settings | KPI 기준 조회 |
| promo_records | 홍보 방문 기록 |

---

## 조회 유형별 처리

### 접수 현황
- "이번 달 접수 몇 건?" → projects WHERE created_at 이번 달 COUNT
- "수원 진행중 건" → projects WHERE city=수원 AND status=진행중
- "미수금 현황" → projects WHERE outstanding > 0 합계
- "XX빌라 어디까지 왔어?" → 해당 건 현재 단계 + 부족 항목

### 일정
- "이번 주 실측 일정" → schedules WHERE type=실측 AND this_week
- "내일 빈 직원" → schedules에 없는 직원 찾기
- "수원 세류동 홍보 언제?" → promo_records WHERE 동=세류동 최신

### 직원 활동
- "김직원 이번 주 활동" → activity_log 집계 (접수 N건, 실측 N건 등)
- 관리자만 다른 직원 조회 가능 / 직원은 본인만

### 지출 분석
- "이번 달 카드 지출 이상치" → expenses에서 건당 기준 분석
  - 식대 1건 15,000원+ → 플래그
  - 편의점 매일 4,000원 패턴 → 플래그
- "호계 현장 지출 얼마?" → expenses WHERE site=호계 합계

## 보고서 생성

### 12개 소스 → 4종 보고서
```
매일: activity_log 요약 + 긴급 알림 (서류 마감 등)
주간: 접수/완료 건수 + 현장 공정 + 직원 성과 + 미수금 + 통합 요약
월간: 카드/고정 지출(새는돈) + KPI + 지원사업 통계 + 통합 요약
현장마감: 수익률(마감기준) + 협력업체 평가 + 소장 소명 반영
```

### 총괄 보고서
- 모든 보고서 합쳐서 우선순위 정렬: 긴급 → 주의 → 정상
- 미확인 보고서 상단 고정
- 관리자 분석 페이지에서 열람

## 알림/제안 로직

### 마감 알림
- 서류 제출 마감 D-7, D-3, D-1 알림
- "시지원금 지연 위험" 등 영향도 같이 안내

### 빈 일정 감지
- 직원 캘린더에 3일 이상 빈 일정 → 홍보/팔로업 제안
- "이직원 4/7~10 비어있어요. 홍보 보낼까요?"

### 홍보 제안
- 미방문 동 + 빈 일정 + 동선 조합
- "부천 107일 미방문 → 김직원 4/17 빈 일정에 부천 홍보 추천"
- 대상: 소규모 15시 / 수도 7시 / 학교 7개 (동 단위)

### 공문 패턴 예측
- 작년 공문 발송 시기 기준 2달 전 알림
- "작년 수원시 소규모 공문 6월 발송 → 4월부터 홍보 준비"

### 시간 교정 (데이터 축적 후)
- 과거 실측 평균 시간 vs 캘린더 입력 시간 비교
- "실측 2시간 잡았는데 평균 42분 → 1시간으로 줄이세요"
- 기준은 데이터 쌓이면서 재호와 조율

## KPI 산출

### 4대 영역 자동 계산
| 영역 | 지표 | 소스 |
|------|------|------|
| 지원사업 | 접수 전환율 | projects |
| 지원사업 | 일정 준수율 | schedules |
| 지원사업 | 서류 오류율 | documents |
| 지원사업 | 미수금 회수 기간 | projects |
| 현장 | 공정 준수율 | site_logs |
| 현장 | 예산 대비 지출 | expenses |
| 현장 | 하자 발생 | A/S 테이블 |
| 업무효율 | 활동량 추이 | activity_log |
| 업무효율 | 홍보 달성률 | promo_records |
| 협력업체 | 납기 준수율 | site_logs |
| 협력업체 | 재고용 횟수 | 거래처DB |

- 정량 = AI 자동 산출
- 정성 = 대표 설정 + KPI 에이전트가 기준 수정 지원
