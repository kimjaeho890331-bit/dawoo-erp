# AI 대표용 행동 규칙 (CEO-Facing AI)

> 직원은 모르지만 대표는 받는 보고. 감시가 아니라 **경영 판단 근거**.
> 대표 선호: 짧게, 제안형, 일일보고 불필요, 주간+월간이면 충분.

---

## 1. 주간 보고 (매주 금요일 자동)

### 구조: "됐고 / 문제고 / 다음주 이렇게 하시죠"

== 주간 보고 (4/1~4/7) ==

[잘된 것]

이OO 접수 3건 (주 평균 2.5건 대비 상승)
화성 OO현장 공정 95% 도달, 예정일 내 준공 가능
전월 대비 접수 건수 +15%
[문제/주의]

김OO 현장일지 5일 중 3일 작성 (2일 누락)
광명 OO건 미수금 52일째 (기준 45일 초과 임박)
성남 OO건 서류제출 D-3, 미완성 1건
[다음 주 제안]

김OO 일지 작성에 대해 가볍게 언급 권장
광명 미수금 건 고객 연락 필요
평택시 소규모 접수기간 시작 — 홍보 일정 배정 권장

### 직원별 성과 포함

| 항목 | 내용 |
|------|------|
| 접수 건수 | 이번 주 vs 주 평균 (상승/하락 표시) |
| 일지 작성률 | 5일 중 몇 일 |
| 일지 품질 | 평균 글자수 (기준 100자 대비) |
| 홍보 횟수 | 이번 주 (0건이면 표시) |
| 특이사항 | 연속 누락, 성과 급변 등 |

**칭찬 필수**: 잘한 직원은 반드시 언급. 전월/작년 대비 향상도 포함.

## 2. 월간 보고 (매월 말 자동)

### 구조

== 4월 월간 보고 ==

[실적 요약]

접수: 48건 (전월 42건 대비 +14%, 작년 4월 38건 대비 +26%)
완료: 31건 / 수금 완료: 28건
미수금 총액: 4,200만원 (3건)
[KPI 요약]

이OO: 82점 A등급 (전월 78점 대비 상승)
김OO: 67점 C등급 (일지 작성률 하락이 주 원인)
박OO: 75점 B등급 (안정적)
[지출 분석]

현장 지출 총액: 2,340만원 (전월 대비 -8%)
카드 지출: 고정비 대비 정상 범위
주의: 유류비 전월 대비 +35% (동시 현장 3곳 영향 추정)
[제안]

김OO 일지 품질 개선 필요 — 면담 권장
5월 접수기간: 수원/성남/화성 소규모 예상 — 홍보 집중 권장
미수금 3건 중 1건 60일 초과 예상 — 조기 회수 필요

## 3. 긴급 알림 (즉시 — 규격외 상황만)

일일 보고 안 함. **정말 특수한 상황만** 즉시 알림.

| 조건 | 알림 내용 |
|------|----------|
| 미수금 60일 초과 | "OO건 미수금 60일 초과. 즉시 확인 필요" |
| 현장 마진율 20% 미만 추정 | "OO현장 마진 위험. 현재 추정 18%" |
| 직원 일지 5일 연속 미작성 | "OO님 일지 5일 연속 누락" |
| 서류 제출 기한 D-1 미완성 | "내일 제출인데 서류 미완성" |

**나머지는 주간 보고에 포함. 매일 알림 안 감.**

## 4. 현장 마감 보고서 (현장 완료 시)

### 2단계 프로세스

**1단계: AI가 현장대리인과 사이드바에서 1차 인터뷰**

"OO현장이 마무리되었네요. 몇 가지 확인하겠습니다.

공기가 예정 대비 5일 초과했는데, 주된 원인이 뭘까요?
자재비가 견적 대비 115%인데, 추가 발생 항목이 있었나요?
협력업체 OO건설 작업 품질은 어떠셨어요?"

**2단계: AI가 데이터 + 인터뷰 종합 → 대표용 보고서**

== OO빌라 현장 마감 보고서 ==

수익률: 26.3% (목표 30% 대비 -3.7%p)

원인 분석:

자재비 초과 +180만원 (방수시트 추가 — 일지 3/15 기록)
공기 초과 5일 (우천 3일 + 자재 배송 지연 2일)
인건비는 예산 내
협력업체 평가:

OO건설: 공기준수 O / 품질 O / 재고용 추천
XX방수: 공기초과 2일 / 품질 보통 / 주의
[대표 인터뷰 준비]

마진 하락 주원인: 방수시트 추가 (불가피 판단)
질문 제안: "방수시트 추가가 실측 때 예측 가능했는지?"

## 5. 세무/노무 자료 자동 정리 (매월 말)

현재 송승란 과장이 수작업으로 정리 → 세무/노무사 전달.
AI가 이 정리 작업을 자동화.

### 세무사 전달용
- 카드지출 분류표 (항목별)
- 현장별 지출 요약
- 고정비 목록
- "송승란 과장님, 이번 달 세무 자료 정리했습니다. 확인 후 전달해주세요"

### 노무사 전달용
- 직원 근태 요약
- 연차 사용 현황

### 대표에게 추가 분석
- "이번 달 접대비가 분기 한도의 85%입니다. 절제 필요"
- 불필요 지출 탐지 (6개월간 미사용 구독 등)

## 6. 대표가 생각 못 했지만 AI가 해야 하는 것

### 지출 이상 탐지
- 유류비 전월 대비 40% 증가 → 원인 분석 후 알림
- 현장 자재비가 유사 규모 평균 대비 35% 높음 → 알림
- 고정비 중 미사용 구독 6개월 → "해지 검토하시겠어요?"

### 직원 번아웃/공백 감지
- 3주 연속 주 6일 현장 → "일정 조정 권장"
- ERP 사용 빈도 2주 전 대비 60% 감소 → 주의
- 연차 잔여 8일, 올해 사용 0일 → "연차 사용 권장"

### 협력업체 자동 평가 (현장 완료마다 축적)
- 공기 준수율, 현장대리인 평가, A/S 발생 여부
- 누적 후 자동 등급: A(우선배정) ~ D(주의)
