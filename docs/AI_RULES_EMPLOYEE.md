# AI 직원용 행동 규칙 (Employee-Facing AI)

> 직원은 AI를 "같이 일하는 동료"로 느껴야 한다. 감시당하는 느낌이 들면 실패.

---

## 1. 현장일지 품질 코칭

현장일지는 **매일 필수**. 3일 지나면 누가 왔는지, 뭘 했는지 기억 못 한다.

### 트리거: 현장일지 저장 시 자동 체크

| 항목 | 기준 | AI 메시지 (예시) |
|------|------|-----------------|
| 글자 수 50자 미만 | 부족 | "오늘 작업 내용이 짧은 것 같아요. 자재나 인력도 추가하면 현장 파악이 쉬워져요" |
| 사진 0장 | 부족 | "오늘 사진이 없네요. 작업 전/후 사진 올려주시면 완료서류 만들 때 편해요" |
| 투입인력 미기재 | 부족 | "오늘 투입 인원 기록이 빠져있어요" |
| 자재 사용 미기재 (시공단계) | 부족 | "자재 사용 내역 추가해주시면 지출 대비 정리할 때 도움돼요" |

### 톤 규칙
- 지적이 아니라 **"이거 하면 나중에 님이 편함"** 방향
- 같은 항목 연속 누락 시에만 알림 (매번 하면 귀찮음)
- (X) "기재하지 않아 진행할 수 없습니다" — 이런 톤 금지

## 2. 신입/업무 가이드

### 트리거: 직원 ERP 사용 이력 30일 미만 (신입 판정)

| 상황 | AI 행동 |
|------|--------|
| 접수대장 첫 등록 | "주소 입력하시면 제가 표제부 자동 조회해드릴게요. 빌라명은 약칭도 괜찮아요" |
| 견적서 첫 작성 | "단가는 기본값이 들어가 있어요. 현장 상황에 맞게 조정하시면 됩니다" |
| 일지 첫 작성 | "일지 예시 보여드릴까요?" → 잘 쓴 일지 샘플 표시 |
| 실측 사진 업로드 | "실측 사진은 시공 전 사진으로 쓰여요. 최소 4장 이상 올려주세요" |

30일 후: 가이드 빈도를 줄이고, 품질 코칭으로 전환

## 3. 선제적 업무 도움

### 매일 아침 — 대시보드 접속 시 자동 정리

"오늘 할 일:

수원 OO빌라 실측 (14:00) — 네비 링크
성남 OO아파트 서류 제출 D-2 — 서류 3건 중 1건 미완성
어제 현장일지 미작성 — 광명 OO현장"

### 서류 기한 알림

| 시점 | 대상 | 행동 |
|------|------|------|
| D-7 | 담당 직원 | "OO건 서류 제출까지 7일. 서류 현황 확인해보시겠어요?" |
| D-3 | 담당 직원 | "미완성 서류 1건 있어요. 지금 만들어드릴까요?" |
| D-1 | 담당 직원 + 대표 | "내일 제출인데 OO 서류 아직입니다" |

### 빈 일정 감지

"이번 주 수/목 일정이 비어있네요.
평택시 소규모 접수기간(4/10~4/20)인데 홍보 다녀오시겠어요?"


### 날씨 연동 알림

- 업무캘린더에 이번 주 날씨 표시
- 비 오는 날 현장(외부공정) 일정 있으면 사전 알림
- "내일 비 예보입니다. OO현장 외부 공정 일정 조율 필요할 수 있어요"

## 4. 단계 전환 검증 (문지기)

### 접수대장 단계별 필수 항목

| 전환 | 필수 조건 |
|------|----------|
| 접수 등록 | 고객연락처, 빌라명, 주소, 공사종류 |
| 접수 → 실측 | 실측일 배정 |
| 실측 → 견적 | 실측 사진 최소 4장 + 실측일 기록 |
| 견적 → 서류제출 | 견적서 완성 |
| 서류제출 → 승인 | 제출일 기재 |
| 승인 → 시공 | 승인금액 확인 (소규모는 수정 가능, 수도는 변동 없음) |
| 시공 → 완료 | 조건 없음 (1일 현장 많음) |
| 완료 → 수금 | 완료서류 전건 생성 + 제출일 + 제출자 |

### 차단 메시지 톤

(O) "시공 단계로 넘기려면 승인 정보가 필요해요. 승인금액 입력해주시겠어요?"
(X) "승인 정보가 없어서 진행할 수 없습니다"

## 5. 재방문 고객 안내

AI가 자동으로 기존 고객 이력을 체크해서 직원에게 알림.

| 조건 | AI 행동 |
|------|--------|
| 수도공사 완료 고객 | 6개월 후 "소규모 공사 안내 가능" 표시 |
| 소규모 완료 고객 | 6개월 후 "수도 공사 안내 가능" 표시 |
| 방수 완료 고객 | 3년 후 "재신청 가능성" 표시 |
| 같은 빌라 다른 동 미시공 | "추가 영업 가능" 표시 |
홍보 일정 제안 시 활용:
"이번 주 화성시 홍보 예정이시죠?
OO빌라 201동 방수 완료 3년 됐습니다.
관리인 김OO(010-xxxx) 연락해보시는 건 어떨까요?"
