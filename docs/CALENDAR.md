# docs/CALENDAR.md - 캘린더 상세 스펙

## 데이터 구조
- DB 테이블: schedules (하나)
- 뷰가 여러 개

## 데이터 흐름

### 입력 4곳
1. 접수대장 → 실측일, 시공일, 제출일
2. 현장관리 → 착공~준공 공정 일정
3. 직원/관리자 → 개인 일정, 회의
4. AI 비서 → 홍보/미팅 일정 제안

### 출력 4곳
1. 업무 캘린더 → 지원사업 + 개인일정 + 홍보 통합
2. 현장관리 캘린더 → 해당 현장 공정만
3. 접수대장 상세 → 해당 건 일정만
4. 대시보드 → 오늘 일정

### AI 4기능
1. 빈 일정 감지 → 홍보/미팅 제안
2. 일정 충돌 체크 → 협력업체 겹침 알림
3. 공문 패턴 예측 → 작년 기준 2달 전 알림
4. 마감 알림 → 제출 D-day

## 업무 캘린더 화면

### 상단
- 직원별 태그 (색깔 구분, 클릭 필터 on/off)
- 월/주/일 뷰 전환

### 캘린더 탭
- 월간 그리드에 일정 바 표시
- 일정 형태: "김 | 권선 실측" (직원 | 업무)
- 색깔 구분: 직원별 (김=파랑, 박=핑크, 이=초록)
- 종류별: 연차=빨강, 홍보=노랑

### AI 업무 교정 패널 (하단 고정)
- **시간 교정**: "2시간 잡았는데 과거 평균 42분. 1시간으로 줄이세요"
- **동선 최적화**: "같은 동네 실측 + 홍보 묶기"
- **빈 일정 감지**: "3일 비어있어, 홍보 or 팔로업 추천"
- **마감 알림**: "완료서류 D-3, 시지원금 지연 위험"
- AI 교정 기준은 데이터 쌓이면서 조율

### 홍보현황 탭
- 소규모/수도 전환
- 시별 동 단위 방문/미방문 프로그레스 바
- 미방문 일수 빨간색 강조
- AI 홍보 제안 (빈 일정 + 미방문 + 동선 조합)
- 대상: 소규모 15개시 / 수도 7개시 / 학교 7개 지역 (동 단위로 많음)
