# 다우건설 AI ERP 시스템 - 전체 설계문서

> 최종 수정: 2026.04.07
> 작성: Claude AI + 김재호 대표
> 이 문서는 전체 시스템의 설계 결정사항을 기록합니다.

---

## 1. 프로젝트 개요

- **목적**: 다우건설 전용 AI 기반 ERP (접수관리 + 현장관리 + 서류 자동화 + 분석)
- **대상**: 정부 지원사업 (15개 시 × 수도/소규모 공사)
- **사용자**: 다우건설 직원 5명 + 대표(김재호)
- **기술 스택**: GitHub + Vercel + Supabase + Claude API
- **입구**: 웹 대시보드 (AI 채팅 사이드 패널) / 텔레그램 (나중 추가)

---

## 2. 핵심 철학

### AI가 주인공, 사람이 서브
- AI가 데이터 입력/서류 생성/판단을 한다
- 사람은 확인/수정/보조를 한다
- AI가 일하기 좋은 구조를 우선한다

### 사람 편의성 최우선
- AI가 아무리 잘해도 사람이 보조할 공간은 직관적이어야 한다
- 대시보드/확인 화면은 사람이 보기 쉽고 수정하기 쉽게
- 1차 개발 없이 전체 설계 완료 후 한번에 제대로 만든다

---

## 3. 시스템 구조

### 에이전트 구조
- 라우터 1개 + 에이전트 3개 (접수/서류/조회)
- 접수 에이전트: DB 등록/수정, 표제부/전유부 연동
- 서류 에이전트: PDF 자동 생성 (폼필드 방식)
- 조회 에이전트: 현황/데이터 검색, 보고서 생성

### 기술 결정
- PDF 자동 입력: 폼필드(Form Field) 방식 (pypdf)
- PDF 한글 렌더링: reportlab + NanumGothic (백업)
- 파일 저장: Supabase Storage
- 동시 사용: 5명 동시 OK (비동기)
- 견적서: 웹 스프레드시트 방식

### 지원사업
- 수도공사 접수대장 (CRM+수금+견적 포함)

### 현장

- 지출결의서
- 연차신청

### 데이터
- 거래처 DB (협력업체 + 일용직 탭 통합)
- A/S 관리

### 세무/회계
- 카드지출 분석
- 고정지출 관리

### 관리
- 서류함 (회사기본/수도/소규모/입찰현장 탭)
- 직원관리
- 회계달력

### 분석
- 보고서 (매일/주간/월간/현장마감)
- KPI

### 공통
- 대시보드 (직원용)
- 공지사항 (규율/규정 포함)
- 설정
- AI 비서 = 오른쪽 사이드 패널 (항상 접근 가능)

## 5. 접수대장 상세 설계

### 목록 화면
- 소규모 / 수도공사 **탭 전환**
- 기본 뷰 = **진행중** (완료/취소 제외)
- 상태 필터: 진행중, 접수, 승인, 완료, 취소, 문의(예약), 전체
- 단계 필터: 없음 (상태로 충분)
- 종류 필터: 없음
- **시 = 클릭 태그** (추가/삭제 가능, 15개 시)
- 목록 컬럼 순서: **담당직원**(맨 앞) → 빌라명 → 소유주 → 연락처 → 주소 → 종류 → 단계 → 상담내역 → 총공사비 → 미수금
- 시 컬럼 없음 (상단 필터에 있으니까)

### 상세 패널 (행 클릭 시)
- 프로그레스 바 (10단계)
- 서류생성 / 수정 / 닫기 버튼

### 상시 표시 영역 (모든 탭에서 항상 보임)
- 담당자, 빌라명, 주소(도로명+지번), 소유주, 연락처
- 공사종류, 현재 단계
- 상담내역/메모
- 총공사비, 자부담금, 시지원금, 추가공사금, 미수금

### 탭 구조
1. **기본정보**: 소유주, 세입자연락처, 동/호/면적/세대수, 사용승인일, 통장정보(드래그앤드롭→OCR자동입력), 진행연도
2. **1단계 접수~신청**: 실측(날짜/담당자/메모/사진), 견적(자부담/시지원/추가공사+견적서 드래그앤드롭), 동의서(수령일+스캔), 신청서(제출일/제출자+신청서/통장사본 드래그앤드롭)
3. **2단계 승인~공사**: 승인일, 시공일, 시공업체/직영, 장비, 착수금, 공사완료일, 시공전/시공중/시공후 사진 드래그앤드롭
4. **3~4단계 완료~수금**: 완료서류(제출일/제출자/보고서 드래그앤드롭), 수금(복수 입금 테이블: 자부담착수금+추가공사비+시지원금잔금)
5. **서류/첨부**: 생성 서류 목록 + 첨부파일 드래그앤드롭
6. **이력**: 단계 변경 타임라인

### 단계 전환 규칙
- AI가 데이터 채워짐에 따라 자동 진행
- 필수 항목 미입력 시 다음 단계 불가 + 부족 항목 안내
- 서류 부족 시 프린트 차단

### 단계별 필요 입력 항목
- 문의→실사: 빌라명, 주소, 소유주, 연락처, 메모, 실측일 예약
- 실사→견적: 실측일, 담당자, 면적, 세대수, 사용승인일, 현장메모, 실측사진(선택)
- 견적→동의서: 견적서 생성, 총공사비 확정, 전달 확인
- 동의서→신청서: 동의서 수령, 스캔 첨부
- 신청서→승인: 신청서 PDF, 제출일, 제출자, 통장사본
- 승인→착공계: 승인일, 시공일, 시공업체/직영
- 착공계→공사: 착공계 제출 (시에 따라 생략)
- 공사→완료서류: 공사 완료일, 전/후 사진
- 완료서류→입금: 완료보고서, 제출일, 제출자
- 입금: 입금일, 입금자, 금액, 미수금=0

### 수금 구조
- 복수 입금 가능
- 자부담금(착수금) + 추가공사비 + 시지원금(잔금) 분리
- 입금 내역 테이블 형태

### 승인 번호: 없음

## 6. 현장관리 상세 설계

### 목록
- 아코디언 방식 (접었다 폈다, 페이지 이동 없음)
- 헤더: 현장명, 상태, 담당자, 공정률 바, 예산

### 펼치면
- **공정 캘린더 상단 고정** (레퍼런스 스타일)
  - 월간 캘린더 그리드에 공종 바 표시
  - 왼쪽 공종 태그 → 캘린더로 드래그앤드롭
  - 바 스트레치로 일정 늘리기/줄이기
  - 드래그앤드롭으로 일정 위치 변경
  - 처음 놓으면 = 테두리만 (미확정)
  - 더블클릭 → 상세 모달 (시작일/종료일, 시공팀 검색, 투입작업자 검색, 메모)
  - 업체/작업자 선택 + 확정 → 색 채워짐 (확정)
  - 캘린더에 "철거 (코끼리철거)" 처럼 공종 + 업체명 표시
  - 공종 추가/삭제 가능
  - 시공팀/투입작업자 = 검색 형태 (100개 이상 가능)
  - 금액 필드 없음

### 탭
1. **기본정보**: 현장 상세 + 예산/지출/잔액/집행률 + 메모
2. **현장일지**: 일지 작성 폼 (날짜/날씨/금일작업/투입인력/자재/특이사항/익일계획)
   - 사진 2칸 구분:
     - 칸1: 현장 사진 4장 고정 (매일 동일 앵글) → 자동 사진대장 생성
     - 칸2: 기타 사진 20장 자유 업로드
   - 월별 자동 폴더링 (3~6개월 장기 현장 대응)
   - 준공 시 AI가 주요 시점 사진 뽑아서 준공 사진첩 자동 생성
3. **지출**: 지출결의서에서 등록된 내역 자동 표시 (뷰만, 데이터 하나)
4. **서류**: 단계별 폴더 (착공/공사중/준공/상시서류/수금/기타)
   - 1줄 3~4개 카드, 드래그앤드롭 가능한 크기
   - 초록 = 완료 / 점선 = 미제출 / "자동" 태그 / "거래처" 태그
   - AI 서류 체크 (긴급/안내 알림)
   - PDF 출력 + 전체 다운로드 버튼

### 현장 보고서 프로세스
1. AI가 현장일지 기반 초안 자동 생성
2. 소장이 본인 의견 + 소명 추가
3. 관리자(재호)와 소장 미팅 → 수정/보완
4. 최종 보고서 확정 → KPI 반영

### 현장 캘린더 (별도 메뉴)
- 불필요. 현장관리 안 개별 캘린더로 충분.

## 7. 견적서 설계

### 방식: 웹 스프레드시트
- 엑셀 수식이 곱하기/나누기/IF 수준이라 웹 구현 가능
- 실시간 수정 가능 (금액 조정 등)

### 구조
- 서류함에 견적서 기본 폼 1개 (15개 시 공통, 시별 차이는 회사정보만)
- 단가표(노임/자재) = 견적서 내부에 포함
- 접수대장에서 "견적서 열기" → 웹 스프레드시트 열림
- 고객 데이터 자동 입력 + 실측 사이즈 넣으면 실시간 계산
- 공사종류 체크 (옥상방수/금속기와/외벽도장/계단실도장/외벽발수)
- PDF/엑셀 출력 가능

### 사람이 넣는 것
- 실측 사이즈 (옥상 가로/세로/수직, 기와폭, 외벽 가로/세로, 계단실 가로/세로, 건물높이)
- 공사종류 체크
- 금액 조정 (필요시)

### 시트 구조
- 고객정보 (접수대장에서 자동)
- 갑지 (출력용 표지)
- 원가계산서 + 집계표
- 수량산출서 (옥상/외벽)
- 내역서 (방수/기와/외벽도장/계단실도장/외벽발수)
- 일위대가표 (기와/도장발수)

### 단가 관리
- 서류함에서 견적서 열어서 단가 수정
- 수정된 단가는 이후 모든 견적서에 반영

## 8. 서류함 설계

### 역할: 빈 양식/템플릿 원본 보관소

### 4개 탭
1. **회사 기본서류**: 사업자등록증, 법인등기부등본, 인감증명서, 건설업면허, 납세증명서, 4대보험, 통장사본 등
   - 유효기간 관리 (AI가 만료 D-day 알림)
   - 접수대장/현장관리에서 자동 연동
   - 지점 없음 (수원 본사만)
2. **수도**: 시별 신청서 템플릿(폼필드) + 견적서(웹 스프레드시트, 단가 내장) + 완료보고서/사진대장 양식
3. **소규모**: 시별 신청서 템플릿 + 견적서(5종 공사) + 완료보고서/사진대장 양식
4. **입찰/현장**: 착공계/안전관리/폐기물 양식 + 준공서류/정산/하자보수 양식 + 계약/하도급 양식 + 사진대장 템플릿

### 서류 자동화 흐름
- 재호가 빈 양식 PDF 올림 → Claude가 폼필드 추가 → 템플릿 생성
- 접수대장에서 "서류 생성" → 서류함에서 최신 템플릿 가져옴 → 고객 데이터 자동 입력
- 신청서 양식 추가 시 제목만 명확하게 넣으면 AI가 폼필드 자동 매칭

### 사진은 서류함이 아닌 현장관리에서 관리
- 서류함에는 사진대장 빈 양식만 보관

## 9. 캘린더 설계

### DB는 하나 (schedules), 뷰가 여러 개

### 데이터 입력 4곳
- 접수대장 (실측/시공/제출 일정)
- 현장관리 (착공~준공 공정)
- 직원/관리자 (개인 일정)
- AI (홍보/미팅 제안)

### 업무 캘린더
- 지원사업 일정 + 직원 개인일정 통합
- 직원별 색깔 구분, 태그 클릭 필터
- 월/주/일 뷰 전환
- **AI 업무 교정 패널 (하단 고정)**:
  - 시간 교정 (과거 데이터 기반)
  - 동선 최적화
  - 빈 일정 감지 → 홍보/팔로업 추천
  - 마감 알림
- **홍보현황 탭**: 시별 동 단위 방문/미방문, AI 홍보 제안
- AI 교정 기준은 데이터 쌓이면서 조율

### AI 캘린더 기능
- 빈 일정에 홍보/미팅 제안
- 일정 충돌 체크 (협력업체 겹침)
- 공문 패턴 예측 (작년 기준 2달 전 알림)
- 마감 D-day 알림

## 10. 대시보드 (직원용)

- 오늘 일정 (캘린더 연동, 시간순, 색깔 분류)
- 내가 받은 업무 (지시자, 마감일, 체크박스, D-day)
- 내가 지시한 업무 (담당자, 마감일, 진행 여부)
- AI 비서 제안 (홍보, 주간리포트, KPI)
- 나의 현황 (숫자 요약)
- 메모장 (고정, 자유 작성)

## 11. 관리자 분석 페이지

### 우선순위: AI 긴급사항 → 사람(직원) → 현장 → 돈

### 5개 탭: 긴급/요약, 직원 성과, 현장 현황, 매출/지출, 보고서 목록

### 보고서 체계
- **매일**: 직원별 활동 요약 (로그 자동), 긴급 알림
- **주간**: 지원사업 현황, 현장 공정, 직원 성과, 미수금, 주간 통합 요약
- **월간**: 카드/고정 지출 분석, KPI 종합, 지원사업 통계, 월간 통합 요약
- **현장 마감 시**: 최종 보고서 + 수익률 + 협력업체 평가
- **분기/연간**: 매출/수익 추이, KPI 종합

### 보고서 표시
- 우선순위 정렬 (긴급→주의→정상)
- 미확인 보고서 상단 유지, 확인하면 아래로

## 12. KPI 설계

### 4대 영역

#### 지원사업 성과 (직원별)
- 접수 전환율: 70% 이상 우수, 50% 이하 개선
- 일정 준수율: 90% 이상 우수, 80% 이하 주의
- 서류 오류율: 5% 이하 정상, 10% 이상 문제
- 미수금 회수 기간: 30일 이내 우수, 60일 초과 주의

#### 현장 성과 (현장별/소장별)
- 공정 준수율: 95% 이상 우수, 85% 이하 주의
- 예산 대비 지출: 100% 이내 우수, 110% 초과 주의
- 하자 발생: 0건 우수, 3건 이상 주의
- 안전/민원: 안전사고 0건 기본

#### 업무 효율 (직원별)
- 활동 로그 기반 업무량 (본인 추이 중심)
- 홍보 달성률: 월 목표 대비 80% 이상 우수
- 캘린더 빈 일정 비율: 30% 이상이면 체크 (참고용)

#### 협력업체/일용직 평가
- 납기 준수율, 시공 품질, 재고용 횟수, 단가 적정성

### 정량 vs 정성
- 정량: AI 자동 산출
- 정성: 재호가 기준 설정, KPI 전문 에이전트가 수정/보완 지원
- 분석 페이지 안에서 KPI 기준 수정 가능

---

## 13. 활동 로그

- 모든 시스템 사용 = 자동 기록 (접수 등록, 견적서 생성, PDF 다운, 상태 변경, 메모, 사진 업로드 등)
- 별도 일일 보고서/확인 버튼 불필요
- 관리자만 전체 열람, 직원은 본인 것만

---

## 14. 지출 이상치 감지

- 건당 기준 (합산 아님)
- 식대 1건 15,000원 이상 플래그
- 편의점 매일 4,000원 패턴 감지
- 현장 수익률: 월별 아닌 마감 기준

---

## 15. AI 판단 복잡도 관리

### 해결 방식
- 자연어 애매함: 애매하면 되물어보게
- 단계 전환: 필수 항목 기준으로 자동
- KPI: 판단 기준을 시스템 프롬프트에 룰로
- 지출 이상치: 건당 기준 명확화
- 홍보 우선순위: 데이터 쌓이면서 조율
- 공문 예측: 여유 있게 2달 전부터, 오차 감안

## 16. 하지 않는 것
- 지점 정보 (수원 본사만)
- 연와조 등 건축 구조 정보
- 같은 주소 중복 차단 안 함 (수도+소규모 동시 가능)
- 맥미니/Cowork 방식
- 입찰정보 수집 (외부 업체 사용)
- 자재단가 별도 페이지 (견적서에 내장)
- 접수대장 상세에서 시 표시 (상단 필터에 있으니까)
- 승인번호
- 현장 캘린더 별도 메뉴 (현장관리 안 캘린더로 충분)
- 1차 개발 (전체 설계 후 한번에)
