# docs/DOCUMENTS.md - 서류함 상세 스펙

## 역할
- 빈 양식/템플릿 원본 보관소
- 사진은 현장관리에서 관리 (서류함엔 빈 양식만)

## 4개 탭

### 1. 회사 기본서류
- 사업자등록증, 법인등기부등본, 인감증명서, 건설업면허
- 납세증명서(국세/지방세), 4대보험 가입자명부
- 산재보험 가입증명서, 고용보험 가입증명원
- 통장사본 (수원본사)
- **유효기간 관리**: AI가 만료 D-day 알림
- 접수대장/현장관리에서 자동 연동
- 지점 없음 (수원 본사만)

### 2. 수도
- 시별 신청서 템플릿 (폼필드 포함, 7개 시)
- 견적서 기본폼 (웹 스프레드시트, 단가 내장)
- 완료보고서 양식
- 사진대장 양식

### 3. 소규모
- 시별 신청서 템플릿 (폼필드 포함, 15개 시)
- 견적서 기본폼 (웹 스프레드시트, 5종 공사, 단가 내장)
- 완료보고서 양식
- 사진대장 양식

### 4. 입찰/현장
- 착공계 양식, 안전관리계획서 양식, 폐기물처리계획 양식
- 준공서류 양식, 정산서 양식, 하자보수확인서 양식
- 공사계약서 양식, 하도급계약서 양식
- 사진대장 템플릿

## 서류 자동화 흐름
1. 재호가 빈 양식 PDF 올림
2. Claude가 폼필드 추가 → 템플릿 생성
3. 재호 확인 → 수정 → 최종 확정
4. Supabase Storage + templates 테이블에 저장
5. 접수대장에서 "서류 생성" → 서류함에서 최신 템플릿 가져옴 → 데이터 자동 입력
6. 신청서 양식 추가 시 제목만 명확하게 → AI가 폼필드 자동 매칭
