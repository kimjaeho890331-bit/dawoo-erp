# 구글드라이브 폴더 구조 (서류 저장)

> 기존 파일서버(Z: Dawoo-server)를 구글드라이브로 이전.
> ERP에서 서류 생성/업로드 시 이 구조로 자동 저장.

## ERP 연동 규칙

### 파일명 규칙
- `[시]_[빌라명]_[동호수]_[서류종류]_[날짜].pdf`
- 예: `수원시_신한빌라_301호_신청서_2026-04-13.pdf`

### 자동 저장 시점
| ERP 동작 | 드라이브 저장 |
|----------|-------------|
| 접수대장 서류 생성 (Cowork) | 해당 빌라 폴더에 PDF 저장 |
| 사진 업로드 (실측/시공전후) | 해당 빌라/사진/ 폴더에 저장 |
| 현장일지 작성 | 현장/현장일지/[년월]/ 에 저장 |
| 통장사본 OCR | 해당 빌라 폴더에 저장 |

### 폴더 자동 생성
- 신규 접수 등록 시 → `지원사업/[시]/[지원사업종류]/[빌라명_동호수]/` 자동 생성
- 현장 등록 시 → `현장/[현장명]/` + 하위 폴더 자동 생성

### DB 연동
- `documents` 테이블에 `drive_file_id`, `drive_url` 컬럼 추가 (Phase 2)
- `attachments` 테이블에도 동일

### 구현 시점
