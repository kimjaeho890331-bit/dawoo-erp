# docs/ESTIMATE.md - 견적서 상세 스펙

## 방식
- 웹 스프레드시트 (엑셀 아님)
- 수식: 곱하기/나누기/IF/SUM/TRUNC/ROUNDDOWN 수준 → 웹 구현 가능

## 기본 폼
- 1개 (15개 시 공통, 시별 차이는 회사정보만)
- 단가표(노임/자재) = 견적서 내부에 포함 (별도 페이지 없음)

## 시트 구조
1. 고객정보 (접수대장에서 자동 입력)
2. 갑지 (출력용 표지, 다우건설 정보 자동)
3. 원가계산서 + 집계표 (자동 계산)
4. 수량산출서 옥상 (실측 사이즈 → 면적 자동)
5. 수량산출서 외벽 (실측 사이즈 → 면적 자동)
6. 방수 내역서 (면적 구간별 단가 자동 적용)
7. 기와 내역서
8. 기와 일위대가
9. 외벽 도장 내역서
10. 계단실 도장 내역서
11. 외벽 발수 내역서
12. 도장 및 발수 일위대가

## 소규모 공사종류 체크
- 옥상방수, 금속기와, 외벽도장, 계단실도장, 외벽발수
- 체크한 공종만 해당 내역서 활성화

## 사람이 넣는 것
- 실측 사이즈: 옥상(가로/세로/수직), 기와폭, 외벽(가로/세로), 계단실(가로/세로), 건물높이
- 공사종류 체크
- 금액 조정 (필요시 단가 직접 수정)

## AI가 넣는 것
- 빌라명, 주소, 동수, 세대수, 사용승인일, 소유주, 연락처
- 다우건설 회사정보 (고정)

## 흐름
1. 접수대장 1단계 → "견적서 열기" 클릭
2. 고객 데이터 자동 입력
3. 실측 사이즈 입력 + 공사종류 체크
4. 실시간 자동 계산
5. 금액 조정 필요시 단가 직접 수정
6. 저장 → 서류함에 저장 + 접수대장 서류탭 연동
7. PDF/엑셀 출력

## 단가 관리
- 서류함에서 견적서 열어서 단가 수정
- 수정된 단가 → 이후 모든 견적서에 반영
- 노임단가 기준연도 표시 (예: 2025년 07월)
