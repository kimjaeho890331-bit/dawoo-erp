# DAWOO ERP - 페이지별 화면 스펙

> 이 문서는 각 페이지에 뭐가 보여야 하는지만 적습니다.
> 개발순서, 기술설명, DB 설명 없음. 순수 화면 스펙만.

---

## 1. 사이드바

### 구조 (7그룹 15메뉴)
왼쪽 고정. 접기/펼치기 가능.

- DAWOO ERP (로고)
- 대시보드
- **지원사업**: 소규모 접수대장 / 수도공사 접수대장
- **현장**: 현장관리
- **업무**: 업무 캘린더 / 지출관리(지출결의서+카드지출+고정지출) / 연차신청
- **데이터**: 거래처 DB / A/S 관리
- **관리**: 서류함 / 직원관리 / 회계달력
- **분석**: 보고서 / KPI
- 공지사항
- 설정

### AI 비서
- 오른쪽 하단 플로팅 버튼 (항상 보임)
- 클릭하면 오른쪽 사이드 패널 열림 (채팅 형태)
- 모든 페이지에서 접근 가능

## 2. 접수대장 (소규모 / 수도공사)

### 상단
- 페이지 제목: "소규모 접수대장" (또는 "수도공사 접수대장")
- 소규모 / 수도공사 **탭 전환** 버튼
- "+ 새 접수" 버튼

### 상태 필터 (가로 나열)
- 진행중(기본) / 접수 / 승인 / 완료 / 취소 / 문의(예약) / 전체
- 진행중 = 완료/취소 제외한 모든 건

### 시 필터 (가로 나열, 클릭 태그)
- 15개: 수원,성남,안양,부천,광명,시흥,안산,군포,의왕,과천,용인,화성,오산,평택,하남
- 클릭하면 활성/비활성 토글
- 추가/삭제 가능

### 목록 테이블
컬럼 순서 (왼쪽부터):
1. **담당직원** (맨 앞)
2. 빌라명
3. 소유주
4. 연락처
5. 주소 (말줄임)
6. 종류 (뱃지)
7. 단계 (뱃지, 색깔로 구분)
8. 상담내역 (말줄임)
9. 총공사비 (오른쪽 정렬)
10. 미수금 (오른쪽 정렬, 있으면 빨간색)

하단에 "총 N건 / 총공사비 N원 / 미수금 N원" 요약

### 상세 패널 (행 클릭 시 오른쪽에 열림)
목록 55% / 상세 45% 비율

#### 상단: 빌라명 + 서류생성 버튼 + 닫기 버튼

#### 프로그레스 바 (10단계)
문의 → 실사 → 견적 → 동의서 → 신청서제출 → 승인 → 착공계 → 공사 → 완료서류제출 → 입금
- 완료 단계 = 색 채움
- 현재 단계 = 강조색
- 미래 단계 = 회색

#### 상시표시 영역 (항상 보임, 스크롤해도 고정)
- 담당자, 빌라명
- 도로명주소, 지번주소
- 소유주, 연락처
- 공사종류 (뱃지), 현재 단계 (뱃지)
- 금액 카드 5개: 총공사비 / 자부담 / 시지원금 / 추가공사금 / 미수금
  - 미수금 있으면 빨간 배경
- 상담내역/메모 (노란 배경, 수정 가능)

#### 6개 탭

**기본정보 탭:**
- 소유주, 세입자 연락처
- 동, 호
- 면적(m²), 세대수
- 사용승인일, 진행연도
- 통장정보: 드래그앤드롭 영역 → OCR → 은행/계좌번호/예금주 자동입력
- 메모

**1단계 탭 (접수~신청서):**
- 실측 섹션: 실측일, 담당자, 현장메모, 실측사진(드래그앤드롭)
- 견적 섹션: 자부담금/시지원금/추가공사금 입력 + "견적서 열기" 버튼 (새 페이지로 이동)
- 동의서 섹션: 수령일, 동의서 스캔(드래그앤드롭)
- 신청서 섹션: 제출일, 제출자, 신청서(드래그앤드롭), 통장사본(드래그앤드롭)

**2단계 탭 (승인~공사):**
- 승인 전이면 흐릿하게 표시
- 승인일, 시공일, 시공업체/직영, 장비, 일용직, 착수금
- 공사 완료일
- 시공전 사진 / 시공중 사진(업체) / 시공후 사진 (각각 드래그앤드롭)
- 실측사진은 여기 없음 (1단계에 있으니까)

**3~4단계 탭 (완료~수금):**
- 완료서류 제출일, 제출자, 완료보고서(드래그앤드롭)
- 수금: 복수 입금 테이블
  - 행 추가 가능
  - 컬럼: 입금유형(자부담착수금/추가공사비/시지원금잔금) / 금액 / 입금일 / 입금자명
  - 하단 요약: 총공사비 / 수금합계 / 미수금

**서류/첨부 탭:**
- AI 생성 서류 목록 (견적서, 신청서, 완료보고서 등)
- 첨부파일 목록 (통장사본, 동의서, 사진 등)
- 드래그앤드롭 업로드 영역

**이력 탭:**
- 단계 변경 타임라인 (날짜, 담당자, 이전→현재 단계)

### 소규모 공사종류
옥상방수, 새빛, 녹색, 공동주택

### 수도 공사종류
옥내수도, 공용수도

## 3. 견적서 (새 페이지)

### 진입
- 접수대장 1단계 탭 > "견적서 열기" 클릭 → /estimate/[project_id] 새 페이지

### 상단
- "← 접수대장" 뒤로가기 버튼
- 빌라명 + 소유주 표시
- 엑셀 다운 / PDF 출력 / 저장 버튼

### 총액 요약 바
- 총공사비 / 시지원(80%) / 자부담 / 세대당 — 실시간 반영

### 시트 탭
- 고객정보 / 갑지 / 원가계산서 / 방수내역서 (등 공종별)

### 웹 스프레드시트 형태
- 엑셀처럼 셀 편집 가능
- 수정 가능한 셀 = 노란 배경
- 수식: 곱하기/나누기/IF/SUM/TRUNC/ROUNDDOWN 수준
- 고객 데이터 = 접수대장에서 자동 입력
- 사람이 넣는 것: 실측 사이즈 + 공사종류 체크 + 금액 조정

### 공사종류 체크
- 옥상방수 / 금속기와 / 외벽도장 / 계단실도장 / 외벽발수
- 체크한 공종만 해당 내역서 활성화

### 단가 관리
- 단가표(노임/자재) = 견적서 내부에 포함
- 서류함에서 단가 수정 → 이후 모든 견적서 반영

## 4. 현장관리

### 핵심
- 접수대장과 **완전 별개** 독립 메뉴
- 접수대장 = 지원사업 고객관리
- 현장관리 = 입찰/수의/기타 건설 현장 프로젝트
- 연동 없음

### 상단
- "현장관리" 제목 + 현장 수
- "+ 현장 등록" 버튼

### 현장 등록 시 입력
- 현장명, 주소, 소장/담당자, 시공업체, 예산, 착공예정일, 준공예정일, 메모

### 목록: 아코디언 방식
- 접었다 폈다 (페이지 이동 없음)
- 헤더: 현장명 + 상태뱃지(착공대기/공사중/완료서류/완료) + 담당자 + 공정률 바 + 예산
- 클릭하면 펼침/접힘

### 펼치면: 공정 캘린더 (상단 고정)
- 월간 캘린더 그리드에 공종 바 표시
- 왼쪽에 공종 태그 목록 (추가/삭제 가능)
  - 기본: 철거, 방수, 미장, 타일, 목공, 전기, 소방, 필름, 도배, 바닥재, 메지, 입주청소
- **드래그앤드롭**: 공종 태그를 캘린더로 드래그 → 일정 생성
- **스트레치**: 바 양끝 잡고 늘리기/줄이기
- **위치 변경**: 캘린더 안에서 드래그
- **미확정 vs 확정**:
  - 처음 놓으면 = 테두리만 (미확정, 점선)
  - 더블클릭 → 상세 모달:
    - 시작일/종료일
    - 시공팀 (검색형태, 업체 100개+ 대응)
    - 투입 작업자 (검색형태)
    - 메모
    - 금액 필드 **없음**
    - 확정 체크박스
  - 확정 누르면 → 색 채워짐 + "공종 (업체명)" 표시
  - 예: "철거 (코끼리철거) 2일"

### 4개 탭

**기본정보 탭:**
- 현장명, 소장/담당, 시공업체, 주소, 착공일, 준공예정
- 예산 / 지출 / 잔액 / 집행률 카드 (4개)
- 메모

**현장일지 탭:**
- "+ 일지 작성" 버튼 → 작성 폼 열림
- 작성 폼: 날짜, 날씨(맑음/흐림/비/눈), 금일작업, 투입인력(직종별), 자재, 특이사항, 익일계획
- **사진 2칸**:
  - 칸1: 현장 사진 4장 고정 (매일 동일 앵글, 앵글1~4 표시) → 일별 사진대장 자동 생성
  - 칸2: 기타 사진 20장 자유 업로드
- 일지 목록: 날짜 + 날씨뱃지 + 작업내용 + 투입인력
- 이슈 있으면 빨간 배경으로 표시
- 월별 자동 폴더링 (장기 현장 대응)
- 준공 시 AI가 주요 시점 사진 뽑아서 준공 사진첩 자동 생성

**지출 탭:**
- 지출결의서에서 등록된 내역 자동 표시 (뷰만)
- 컬럼: 날짜 / 항목 / 금액 / 등록자
- 하단: 합계
- "지출결의서에서 등록한 내역이 자동으로 표시됩니다" 안내

**서류 탭:**
- 단계별 섹션: 착공 / 공사중 / 준공 / 상시서류 / 수금 / 기타
- 각 섹션: 1줄 3~4개 카드
- 카드 상태:
  - 초록 배경 = 완료 (파일 있음)
  - 점선 테두리 = 미제출 (드래그앤드롭 대기)
  - "자동" 태그 = AI가 데이터 기반 생성
  - "거래처" 태그 = 거래처 DB에서 연동
- 미래 단계 = 흐릿하게 (준공 단계인데 아직 공사중이면)
- AI 서류 체크: 하단에 긴급/안내 알림 (빠진 서류 감지)
- PDF 출력 + 전체 다운로드 버튼
- 서류 드래그앤드롭 업로드

## 5. 업무 캘린더

### 상단
- 직원별 태그 (색깔 구분, 클릭 필터 on/off)
  - 직원마다 고유 색깔 (김=파랑, 박=핑크, 이=초록 등)
- 월/주/일 뷰 전환

### 2개 탭: 캘린더 / 홍보현황

### 캘린더 탭
- 월간 캘린더 그리드
- 일정 형태: "김 | 권선 실측" (직원 | 업무)
- 색깔 구분:
  - 직원별 고유색
  - 연차 = 빨강
  - 홍보 = 노랑
- 일정 클릭 → 상세/수정

### AI 업무 교정 패널 (캘린더 아래 고정)
- 카드 형태로 제안 표시
- **시간 교정**: "2시간 잡았는데 과거 평균 42분. 1시간으로 줄이세요" + [수정] 버튼
- **동선 최적화**: "같은 동네 실측+홍보 묶기" + [추가] 버튼
- **빈 일정 감지**: "3일 비어있어. 홍보 or 팔로업 추천" + [제안 보기] 버튼
- **마감 알림**: "완료서류 D-3. 시지원금 지연 위험" + [알림 보내기] 버튼
- AI 교정 기준은 데이터 쌓이면서 조율

### 홍보현황 탭
- 소규모 / 수도 전환
- 시별 행:
  - 시 이름 / 동 방문현황 프로그레스 바 / 방문동 리스트 / 미방문동 리스트 / 방문 수 / 미방문 수
  - 미방문 일수 오래된 건 빨간색 강조 ("전체 미방문 107일")
- 대상: 소규모 15개시 / 수도 7개시 / 학교 7개 (동 단위)
- AI 홍보 제안: 하단에 "부천 107일 미방문 → 김직원 빈 일정에 추천" 등

## 6. 서류함

### 역할
- 빈 양식/템플릿 원본 보관소
- 사진은 현장관리에서 관리 (서류함엔 빈 양식만)

### 4개 탭

**회사 기본서류 탭:**
- 필수 서류 카드: 사업자등록증, 법인등기부등본, 인감증명서(대표1/대표2), 사용인감계, 건설업등록증, 납세증명서(국세/지방세), 4대보험 가입자명부, 산재보험 가입증명서, 고용보험 가입증명원, 산재요양 확인서
- 유효기간 관리: "발급일로부터 3개월" / "발급일로부터 30일" / "년 1회 갱신"
- AI가 만료 D-day 알림
- 통장사본 (수원 본사)
- 지점 없음 (수원 본사만)

**수도 탭:**
- 시별 신청서 템플릿 (폼필드 포함, 7개 시)
- 견적서 기본폼 (웹 스프레드시트, 단가 내장)
- 완료보고서 양식
- 사진대장 양식

**소규모 탭:**
- 시별 신청서 템플릿 (폼필드 포함, 15개 시)
- 견적서 기본폼 (웹 스프레드시트, 5종 공사, 단가 내장)
- 완료보고서 양식
- 사진대장 양식

**입찰/현장 탭:**
- 착공 서류 양식: 착공계, 안전관리계획서, 폐기물처리계획
- 준공 서류 양식: 준공서류, 정산서, 하자보수확인서
- 계약 서류 양식: 공사계약서, 하도급계약서
- 사진대장 템플릿

### 서류 추가
- 신청서 양식 추가 시 제목만 명확하게 넣으면 AI가 폼필드 자동 매칭
- 각 섹션에 드래그앤드롭 업로드 영역

## 8. 관리자 분석 페이지

### 우선순위: AI 긴급사항 → 사람(직원) → 현장 → 돈

### 5개 탭
1. **긴급/요약**: AI가 잡은 긴급사항 + 전체 요약
2. **직원 성과**: 직원별 활동 로그 기반
3. **현장 현황**: 현장별 공정률, 예산, 이슈
4. **매출/지출**: 카드지출 분석, 새는돈 감지, 고정지출
5. **보고서 목록**: 매일/주간/월간/현장마감 보고서

### 보고서
- 매일: 직원 활동 요약 + 긴급 알림
- 주간: 지원사업 접수/완료 건수 + 현장 공정 + 직원 성과 + 미수금 + 통합
- 월간: 카드/고정 지출(새는돈) + KPI + 지원사업 통계 + 통합
- 현장 마감: 수익률(마감기준) + 협력업체 평가 + 소장 소명

### 총괄
- 모든 보고서 합쳐서 우선순위 정렬: 긴급 → 주의 → 정상

## 9. KPI

### 4대 영역

**지원사업 성과 (직원별):**
- 접수 전환율: 70%+ 우수, 50%- 개선
- 일정 준수율: 90%+ 우수, 80%- 주의
- 서류 오류율: 5%- 정상, 10%+ 문제
- 미수금 회수 기간: 30일- 우수, 60일+ 주의

**현장 성과 (현장별/소장별):**
- 공정 준수율: 95%+ 우수, 85%- 주의
- 예산 대비 지출: 100%- 우수, 110%+ 주의
- 하자 발생: 0건 우수, 3건+ 주의
- 안전/민원: 안전사고 0건 기본

**업무 효율 (직원별):**
- 활동 로그 기반 업무량 (본인 추이 중심)
- 홍보 달성률: 월 목표 대비 80%+
- 캘린더 빈 일정 비율: 30%+ 체크 (참고용)

**협력업체/일용직 평가:**
- 납기 준수율, 시공 품질(A/S 빈도), 재고용 횟수, 단가 적정성

### 정량 vs 정성
- 정량 = AI 자동 산출
- 정성 = 대표가 기준 설정
- KPI 전문 에이전트가 분석 페이지 안에서 기준 수정/보완 가능

---

## 10. 활동 로그

- 모든 시스템 사용 = 자동 기록
  - 접수 등록, 견적서 생성, PDF 다운, 상태 변경, 메모, 사진 업로드 등
- 별도 보고서 작성/확인 버튼 불필요
- 관리자만 전체 직원 열람
- 직원은 본인 것만 볼 수 있음
- KPI, 보고서에 자동 연동

## 11. 지출 이상치 감지

- 건당 기준 (합산 아님)
- 식대 1건 15,000원+ 플래그
- 편의점 매일 4,000원 패턴 감지
- 현장 수익률: 월별 아닌 마감 기준

## 12. 나머지 페이지 (기본 사항)

### 지출관리 (3개 탭 통합)
- **지출결의서 탭**: 일자별 지출 등록, 현장별 연결 (현장관리 지출탭에 자동 표시), 영수증 사진 첨부
- **카드지출 탭**: 직원별 카드 CSV 업로드 → AI 분석, 새는돈 감지
- **고정지출 탭**: 매월 고정비 관리 (임대료, 보험, 차량 등)

### 연차신청
- 신청서 폼 + 승인/반려

### 거래처 DB
- 협력업체 탭 + 일용직 탭 (통합 메뉴, 위에서 탭 전환)
- 업체명, 대표자, 전화, 사업자번호, 전문분야, 통장정보, 평가점수

### A/S 관리
- 신고일, 이슈내용, 해결일, 해결내용, 상태(접수/진행중/완료)
- 접수대장 or 현장관리 연동 가능

### 회계달력
- 세금 납부일, 보험료, 급여일 등 일정 관리

### 직원관리
- 직원 목록, 연락처, 역할, 캘린더 색깔

### 공지사항
- 회사 규율/규정 포함
- 공지 등록/수정/삭제

## 하지 않는 것

- 승인번호 없음
- 지점 없음 (수원 본사만)
- 현장 캘린더 별도 메뉴 없음 (현장관리 안 캘린더로 충분)
- 같은 주소 중복 차단 안 함 (수도+소규모 동시 가능)
- 단가표 별도 페이지 없음 (견적서 내장)
- 접수대장에 시 컬럼 없음 (상단 태그 필터에 있으니까)
- 접수대장과 현장관리는 연동 없음 (완전 별개)
