# 페이지별 화면 스펙
> 각 페이지에 뭐가 보여야 하는지만 적음. 기술/DB 설명 없음.

## 1. 사이드바
왼쪽 고정, 접기/펼치기. 7그룹 15메뉴.
AI 비서: 오른쪽 하단 플로팅 → 사이드 패널 (채팅, 모든 페이지)

## 2. 접수대장 (소규모/수도)
**상단**: 페이지 제목 + 소규모/수도 탭 + "+새 접수"
**상태 필터**: 진행중(기본)/접수/승인/완료/취소/문의(예약)/전체
**시 필터**: 15개 시 클릭 태그 (추가/삭제 가능)
**목록 컬럼**: 담당직원(맨앞) → 빌라명 → 호수 → 연락처 → 주소 → 지원사업 → 상담내역 → 단계(미수금 빨간표시)
**하단**: "총 N건 / 총공사비 N원 / 미수금 N원"

### 상세 패널 (행 클릭 → 오른쪽 열림)
**프로그레스 바**: 문의→실사→견적→동의→신청→승인→착공→공사→완료→입금 (10단계)
**상시표시 (접기/펼치기)**:
- 빌라명+동, 담당자, 주소, 소유주, 연락처, 지원사업, 현재단계
- 금액 5개: 총공사비/자부담/시지원/추가공사금/미수금
- 상담내역 (인라인 편집)

### 6개 탭
**기본정보**: 소유주/세입자 연락처, 소유자조회(건축물대장), 동/호/면적/세대수/사용승인일, 통장(드래그앤드롭→OCR)
**1단계 (접수~신청)**: 실측(일시+담당자+메모+사진), 견적(금액+견적서열기), 동의서(수령일+스캔), 신청서(제출일+제출자)
**2단계 (승인~공사)**: 승인일, 시공일, 시공업체/직영, 장비, 착수금, 시공전/중/후 사진
**3~4단계 (완료~수금)**: 완료서류(제출일/제출자), 수금(복수입금 테이블: 유형/금액/입금일/입금자)
**서류/첨부**: AI생성 서류 + 첨부파일 + 드래그앤드롭
**이력**: 단계 변경 타임라인
**입금내역**: 1~3단계 전 탭 하단에 공통 표시

### 단계 자동전환
데이터 채워짐에 따라 자동 진행. 동의서/승인은 수동. approval_date는 단계판단에서 제외.

## 3. 견적서 (새 페이지 /estimate/[id])
접수대장 1단계 → "견적서 열기" → 웹 스프레드시트
**상단**: ←뒤로가기 + 빌라명/소유주 + 엑셀/PDF/저장
**총액 바**: 총공사비/시지원(80%)/자부담/세대당
**시트탭**: 고객정보/갑지/원가계산서/내역서(공종별)
**수정가능 셀**: 노란 배경. 실측사이즈+공사종류체크+금액조정
**공사종류**: 옥상방수/금속기와/외벽도장/계단실도장/외벽발수
**단가**: 견적서 내부 포함, 서류함에서 수정→이후 모든 견적 반영

## 4. 현장관리
접수대장과 **완전 별개**. "+현장등록" 버튼으로 독립 생성.
**아코디언**: 헤더(현장명+상태+담당자+공정률+예산) 클릭 → 펼침/접힘

### 공정 캘린더 (펼치면 상단 고정)
월간 그리드 + 공종 바. 왼쪽 공종태그 → 캘린더로 드래그앤드롭.
미확정=점선, 더블클릭→상세모달(시작/종료/시공팀/작업자/메모/확정체크), 확정→색채움+"공종(업체명)"

### 4개 탭
**기본정보**: 현장 상세 + 예산/지출/잔액/집행률 카드
**현장일지**: 날짜/날씨/작업/인력/자재/특이사항. 사진2칸(고정4장+기타20장). 월별 폴더링.
**지출**: 지출결의서 연동 (뷰만)
**서류**: 단계별 카드(착공/공사/준공/상시/수금/기타). 초록=완료, 점선=미제출, "자동"/"거래처"태그

## 5. 업무 캘린더
**상단**: 직원별 색깔 태그 (필터) + 월/주/일 뷰
**일정 바**: 36px높이, 2줄(이름동그라미+제목 / 주소). 셀 최소116px.
**모달**: 확정→완료. 담당자 복수체크(칩UI, 각각 별도일정). 기본 confirmed=false.

### 홍보현황 탭
3탭: 소규모(15시)/수도(7시)/학교
시→동 아코디언. 방문/미방문 프로그레스. 미방문 오래되면 빨간 강조.
**핵심**: 누가 언제 갔는지 관리

## 6. 서류함
빈 양식/템플릿 보관소. 4탭:
- **회사기본**: 사업자등록증, 인감, 건설업등록, 납세, 4대보험 등. 유효기간 관리.
- **수도**: 7개시 신청서 + 견적서 + 완료보고서
- **소규모**: 15개시 신청서 + 견적서(5종) + 완료보고서
- **입찰/현장**: 착공/준공/계약 양식

## 7. 대시보드 (직원용)
오늘일정(캘린더연동) / 받은업무 / 지시한업무 / 나의현황(숫자) / 메모장

## 8. 관리자 분석 페이지
우선순위: AI긴급 → 사람 → 현장 → 돈
**5탭**: 긴급/요약 | 직원성과 | 현장현황 | 매출/지출 | 보고서목록
**보고서**: 매일(활동+긴급) / 주간(지원+현장+직원+미수금) / 월간(지출+KPI) / 현장마감(수익률+협력업체)
**총괄AI**: 긴급→주의→정상 정렬, 미확인 상단 고정

## 9. KPI
4대영역 100점: 지원사업30 + 현장25 + 업무효율25 + 협력업체10 + 대표상황점수10
등급: S(90+) A(80+) B(70+) C(60+) D(60-)
정량75(AI자동) + 정성25(대표직접)
직원 대시보드에서 본인 KPI만 확인 (다른 직원 안 보임)

## 10. 나머지 페이지
**지출관리**: 지출결의서(일자별,현장연결) + 카드지출(CSV→AI분석) + 고정지출(월고정비)
**연차**: 신청+승인/반려
**거래처DB**: 협력업체+일용직 2탭
**A/S**: 신고일/이슈/해결일/상태
**회계달력**: 세금/보험/급여 일정
**직원관리**: 목록+연락처+역할+색깔
**공지사항**: 회사 규율/규정 포함

## 활동 로그
모든 시스템 사용 = 자동 기록. 별도 보고서 불필요.
관리자만 전체 열람, 직원은 본인만.

## 지출 이상치
건당 기준(합산 아님). 식대 1건 15,000원+ / 편의점 매일 4,000원 패턴 / 현장수익률 마감기준.
