# 접수대장 상세 스펙

## 목록 화면
- 소규모/수도 탭 전환 + "+새 접수"
- 상태필터: 진행중(기본)/접수/승인/완료/취소/문의(예약)/전체
- 시필터: 15개 시 클릭태그
- 컬럼: 담당직원(맨앞)→빌라명→호수→연락처→주소→지원사업→상담내역→단계(미수금빨강)

## 상세 패널 (행 클릭)
프로그레스바 10단계. 버튼: 수정/삭제/닫기
**상시표시(접기/펼치기)**: 빌라명+동, 담당자, 주소, 소유주, 연락처, 지원사업, 단계, 금액5개, 상담내역

### 6개 탭
1. **기본정보**: 소유주/세입자, 소유자조회(건축물대장), 동/호/면적/세대수/사용승인일, 통장(드래그앤드롭→OCR)
2. **1단계**: 실측(일시+담당+메모+사진) + 견적(금액+견적서열기) + 동의서(수령일+스캔) + 신청서(제출일/제출자)
3. **2단계**: 승인일, 시공일, 업체/직영, 장비, 착수금, 시공전/중/후 사진
4. **3~4단계**: 완료서류(제출일/제출자), 수금(복수입금: 유형/금액/입금일/입금자)
5. **서류/첨부**: AI서류목록 + 첨부파일 + 드래그앤드롭
6. **이력**: 단계변경 타임라인
- **입금내역**: 1~3단계 전 탭 하단 공통 표시

## 단계 자동전환
데이터 채워짐에 따라 자동. 동의서/승인은 수동. approval_date 단계판단 제외.
자동저장 2초 debounce, onRefresh 호출 안 함 (입력 중 데이터 보존).

## 공사종류
- 소규모 지원사업: 옥상방수, 새빛, 녹색, 공동주택, 개인
- 수도 지원사업: 옥내, 공용
