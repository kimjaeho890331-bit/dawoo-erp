# docs/REGISTER.md - 접수대장 상세 스펙

## 목록 화면

### 탭
- 소규모 / 수도공사 탭 전환

### 필터
- 상태: 진행중(기본), 접수, 승인, 완료, 취소, 문의(예약), 전체
- 시: 클릭 태그 (추가/삭제 가능, 15개 시)
- 단계 필터 없음, 종류 필터 없음

### 목록 컬럼 (순서 중요)
담당직원(맨앞) → 빌라명 → 소유주 → 연락처 → 주소(말줄임) → 종류(뱃지) → 단계(뱃지) → 상담내역(말줄임) → 총공사비 → 미수금

## 상세 패널 (행 클릭 시)

### 상단
- 프로그레스 바 (10단계: 문의→실사→견적→동의서→신청서제출→승인→착공계→공사→완료서류→입금)
- 버튼: 서류생성 / 수정 / 닫기

### 상시 표시 영역 (모든 탭에서 항상 보임)
- 담당자, 빌라명, 도로명주소, 지번주소
- 소유주, 연락처, 공사종류, 현재 단계
- 상담내역/메모 (수정 가능)
- 총공사비, 자부담금, 시지원금, 추가공사금, 미수금

### 6개 탭

#### 1. 기본정보
- 소유주, 세입자 연락처, 동/호/면적/세대수
- 사용승인일, 진행연도
- 통장정보: 드래그앤드롭 → OCR → 은행/계좌/예금주 자동입력

#### 2. 1단계 (접수~신청서)
- **실측**: 날짜, 담당자, 면적결과, 현장메모, 실측사진(드래그앤드롭)
- **견적**: 자부담금, 시지원금, 추가공사금 입력 + 견적서(AI생성 or 드래그앤드롭)
- **동의서**: 수령일, 동의서 스캔(드래그앤드롭)
- **신청서**: 제출일, 제출자, 신청서(AI생성 or 드래그앤드롭), 통장사본(드래그앤드롭)

#### 3. 2단계 (승인~공사)
- 승인일, 시공일, 시공업체/직영, 장비/일용직, 착수금
- 공사 완료일
- 시공전/시공중/시공후 사진 (각각 드래그앤드롭)
- 승인 전에는 흐릿하게 표시
- 실측사진은 1단계에 있으니 여기 없음

#### 4. 3~4단계 (완료~수금)
- 완료서류 제출일, 제출자, 완료보고서(AI생성 or 드래그앤드롭)
- 수금: 복수 입금 테이블
  - 자부담금(착수금) + 추가공사비 + 시지원금(잔금)
  - 입금일, 입금자명, 금액
  - 총공사비/수금합계/미수금 요약

#### 5. 서류/첨부
- 생성 서류 목록 (견적서/신청서/완료보고서 + AI태그 + 미리보기)
- 첨부파일 (통장사본/동의서/실측사진/시공전후 + 드래그앤드롭)

#### 6. 이력
- 단계 변경 타임라인 (날짜, 담당자, 이전→현재 단계)

## 단계 전환 규칙

### 필수 입력 항목 (미입력 시 다음 단계 불가)
- 문의→실사: 빌라명, 주소, 소유주, 연락처, 메모, 실측일예약
- 실사→견적: 실측일, 담당자, 면적, 세대수, 사용승인일, 현장메모
- 견적→동의서: 견적서 생성, 총공사비 확정, 전달확인
- 동의서→신청서: 동의서 수령, 스캔첨부
- 신청서→승인: 신청서 PDF, 제출일, 제출자, 통장사본
- 승인→착공계: 승인일, 시공일, 시공업체/직영
- 착공계→공사: 착공계 제출 (시에 따라 생략)
- 공사→완료서류: 공사 완료일, 전/후 사진
- 완료서류→입금: 완료보고서, 제출일, 제출자
- 입금: 입금일, 입금자, 금액, 미수금=0

### AI 자동 진행
- 데이터 채워지면 AI가 자동으로 단계 전환
- 수동 단계 변경 버튼도 있음 (관리자용)
