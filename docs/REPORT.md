## 데이터 흐름

### 12개 데이터 소스
1. 접수대장 (접수/완료 건수)
2. 현장관리 (공정률/일지/사진)
3. 캘린더 (일정 준수율)
4. 활동 로그 (직원별 시스템 사용)
5. 지출결의서 (현장별/항목별 지출)
6. 카드지출 (직원별 CSV 분석)
7. 고정지출 (매월 고정비)
8. 거래처DB (협력업체 평가)
9. 홍보현황 (지역별 방문 기록)
10. A/S관리 (하자 패턴/빈도)
11. 미수금 (완료건 기준)
12. 현장 보고서 (소장 미팅 후 최종)

### 생성되는 보고서
- **매일**: 직원별 활동 요약(로그 자동) + 긴급 알림
- **주간**: 지원사업 현황 + 현장 공정 + 직원 성과 + 미수금 + 통합 요약
- **월간**: 카드/고정 지출 분석(새는돈 찾기) + KPI 종합 + 지원사업 통계 + 통합 요약
- **현장 마감 시**: 최종 보고서 + 수익률(마감기준) + 협력업체 평가
- **분기/연간**: 매출/수익 추이 + KPI 종합

- 모든 보고서 합쳐서 우선순위 정렬: 긴급 → 주의 → 정상

## KPI

### 4대 영역

#### 1. 지원사업 성과 (직원별)
- 접수 전환율: 70%+ 우수, 50%- 개선
- 일정 준수율: 90%+ 우수, 80%- 주의
- 서류 오류율: 5%- 정상, 10%+ 문제
- 미수금 회수 기간: 30일- 우수, 60일+ 주의

#### 2. 현장 성과 (현장별/소장별)
- 공정 준수율: 95%+ 우수, 85%- 주의
- 예산 대비 지출: 100%- 우수, 110%+ 주의 (소장 소명 반영)
- 하자 발생: 0건 우수, 3건+ 주의 (공종별 패턴)
- 안전/민원: 안전사고 0건 기본

#### 3. 업무 효율 (직원별)
- 활동 로그 기반 업무량 (본인 추이 중심)
- 홍보 달성률: 월 목표 대비 80%+
- 캘린더 빈 일정 비율: 30%+ 체크 (참고용)

#### 4. 협력업체/일용직 평가
- 납기 준수율
- 시공 품질 (A/S 빈도 연동)
- 재고용 횟수
- 단가 적정성

### 정량 vs 정성
- 정량: AI 자동 산출
- 정성: 대표가 기준 설정
- KPI 전문 에이전트: 분석 페이지 안에서 기준 수정/보완 가능

### 지출 이상치
- 건당 기준 (합산 아님)
- 식대 1건 15,000원+ 플래그
- 편의점 매일 4,000원 패턴 감지
- 현장 수익률: 월별 아닌 마감 기준
