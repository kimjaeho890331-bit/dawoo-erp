## 2. 보고서 4종
**매일** (오후5시 자동): 직원 활동요약 + 긴급알림(서류D-3,미수금30일+,공정지연)
**주간** (월요일): 지원사업(접수/완료) + 현장(공정) + 직원(활동) + 미수금 + 홍보
**월간** (1일): 지출분석(새는돈) + KPI종합 + 지원사업통계
**현장마감** (완료시): 수익률(마감기준) + 견적vs실제 + 협력업체평가 + 소장소명

### 현장마감 보고서 프로세스
1. AI가 현장일지 기반 초안 → 2. 소장 의견+소명 → 3. 관리자 미팅 → 4. 최종 확정→KPI 반영

## 3. 총괄 AI (보고서 정리)
모든 보고서 → 우선순위 정렬 → 관리자 분석 페이지 5탭
- **긴급**(빨강): 공기지연, 미수금60일+, 서류D-1, 안전사고, 이상지출
- **주의**(노랑): 미수금30일+, 공정5일+지연, KPI하락, 미방문90일+
- **정상**(초록): 이상 없음

## 4. KPI 배점 (100점)
| 영역 | 배점 | 방식 |
|------|------|------|
| 지원사업 성과 (직원별) | 30점 | 정량 AI |
| 현장 성과 (현장별/소장별) | 25점 | 정량 AI |
| 업무 효율 (직원별) | 25점 | 정량AI+정성대표 |
| 협력업체/일용직 | 10점 | 정량 AI |
| 업무 상황점수 (대표설정) | 10점 | 정성 대표 |

### 4-1. 지원사업 (30점)
| 지표 | 배점 | 우수 | 보통 | 개선 |
|------|------|------|------|------|
| 접수 전환율 | 10점 | 70%+ | 50~69% | 50%- |
| 일정 준수율 | 8점 | 90%+ | 80~89% | 80%- |
| 서류 오류율 | 6점 | 5%- | 5~10% | 10%+ |
| 미수금 회수 | 6점 | 30일- | 30~60일 | 60일+ |

### 4-2. 현장 (25점)
| 지표 | 배점 | 우수 | 보통 | 개선 |
|------|------|------|------|------|
| 공정 준수율 | 8점 | 95%+ | 85~94% | 85%- |
| 예산 대비 지출 | 7점 | 100%- | 100~110% | 110%+ |
| 하자 건수 | 5점 | 0건 | 1~2건 | 3건+ |
| 현장일지 성실도 | 5점 | 매일 | 주3회+ | 주3회- |

### 4-3. 업무 효율 (25점)
| 지표 | 배점 | 방식 |
|------|------|------|
| 활동량 추이 | 8점 | 정량 (본인 전월 대비) |
| 홍보 달성률 | 7점 | 정량 (월목표 80%+) |
| 빈 일정 비율 | 5점 | 정량 (참고용) |
| 업무 태도/협업 | 5점 | 정성 (대표 평가) |

### 4-4. 협력업체 (10점)
납기준수(4점) + 시공품질/A·S(3점) + 재고용(3점)

### 4-5. 업무 상황점수 (10점, 대표 직접)
긴급대응력(3) + 팀기여도(3) + 자기개발(2) + 고객관리(2)
5단계: 탁월100% / 우수80% / 보통60% / 미흡40% / 부족20%
