# 현장관리 상세 스펙

## 핵심
- **접수대장과 완전 별개** (연동 없음, DB도 별개)
- 접수대장 = 정부 지원사업 고객관리 / 현장관리 = 입찰/수의 건설 현장 프로젝트

## 현장 등록
"+현장등록" 버튼으로 독립 생성
입력: 현장명, 주소, 소장/보조, 발주처 담당자/연락처, 착공예정일, 준공예정일, 메모
상태: 계약→착공→공사중→준공서류→정산완료

## 목록
아코디언 방식 (페이지이동 없음). 헤더: 현장명+상태+담당자+공정률+예산

## 공정 캘린더 (펼치면 상단 고정)
- 월간 그리드 + 공종 바. 왼쪽 공종태그 → 캘린더로 드래그앤드롭
- 바 스트레치(늘리기/줄이기) + 드래그(위치변경)
- 미확정=점선테두리, 더블클릭→모달(시작/종료/시공팀검색/작업자/메모/확정체크)
- 확정→색채움+"공종(업체명)" 표시. 금액필드 없음.

## 4개 탭
1. **기본정보**: 현장상세 + 예산/지출/잔액/집행률 카드 + 메모
2. **현장일지**: 날짜/날씨/작업/인력/자재/특이사항/익일계획
   - 사진2칸: 고정4장(동일앵글)→사진대장 자동 + 기타20장
   - 월별 폴더링. 준공시 AI가 주요시점 사진첩 자동생성
3. **지출**: 지출결의서 연동 (뷰만)
4. **서류**: 단계별 카드(착공/공사/준공/상시/수금/기타). 초록=완료, 점선=미제출

## 현장 보고서 프로세스
AI 초안 → 소장 의견+소명 → 관리자 미팅 → 최종 확정 → KPI 반영
