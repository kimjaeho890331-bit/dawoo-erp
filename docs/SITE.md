## 공정 캘린더 (상단 고정)

### 형태
- 월간 캘린더 그리드에 공종 바 표시
- 왼쪽에 공종 태그 목록 (추가/삭제 가능)

### 드래그앤드롭
- 공종 태그를 캘린더로 드래그 → 일정 생성
- 캘린더 안에서 드래그 → 일정 위치 변경
- 바 양끝 스트레치 → 일정 늘리기/줄이기

### 미확정 vs 확정
- 처음 놓으면: 테두리만 (미확정)
- 더블클릭 → 상세 모달:
  - 시작일/종료일
  - 시공팀 (검색형태, 100개+)
  - 투입 작업자 (검색형태)
  - 메모
  - 금액 필드 없음
- 업체/작업자 선택 + 확정 → 색 채워짐 (확정)
- 캘린더에 "철거 (코끼리철거)" 형태로 공종 + 업체명 표시

## 4개 탭

### 1. 기본정보
- 현장명, 소장/담당, 시공업체, 주소, 착공일, 준공예정
- 예산/지출/잔액/집행률 카드
- 메모

### 2. 현장일지
- 작성 폼: 날짜, 날씨, 금일작업, 투입인력(직종별), 자재, 특이사항, 익일계획
- **사진 2칸**:
  - 칸1: 현장 사진 4장 고정 (매일 동일 앵글) → 일별 사진대장 자동 생성
  - 칸2: 기타 사진 20장 자유 업로드
- 월별 자동 폴더링 (장기 현장 대응)
- 준공 시 AI가 주요 시점 사진 뽑아서 준공 사진첩 자동 생성

### 3. 지출
- 지출결의서에서 등록된 내역 자동 표시 (뷰만, 데이터 하나)
- 날짜/항목/금액/등록자 테이블
- 합계 표시

### 4. 서류
- 단계별 폴더: 착공 / 공사중 / 준공 / 상시서류 / 수금 / 기타
- 1줄 3~4개 카드
- 상태: 초록=완료 / 점선=미제출 / "자동"태그=AI생성 / "거래처"태그=DB연동
- 미래 단계 = 흐릿
- AI 서류 체크: 긴급/안내 알림 (빠진 서류 감지)
- PDF 출력 + 전체 다운로드 버튼
- 드래그앤드롭 업로드

2. 소장이 본인 의견 + 소명 추가 (견적오차, 공기지연 사유 등)
3. 관리자와 소장 미팅 → 수정/보완
4. 최종 보고서 확정 → KPI 반영
